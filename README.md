# 时间环 - 智能穿戴产品展示页面 ⏰✨

## 项目简介

这是一个展示智能护腕产品"时间环"的网页项目。页面展示了一只手臂上佩戴着白色篮球护腕，护腕上有一个圆形屏幕显示时间和日期。

## 功能特点

- 🕐 **实时时间显示** - 屏幕显示当前时间和日期
- 💫 **发光动画效果** - 屏幕有呼吸灯效果
- 🎯 **交互体验** - 点击屏幕有反馈效果
- 📱 **响应式设计** - 适配手机和电脑
- 🎨 **真实材质效果** - 护腕和屏幕都有立体感

## 使用方法

1. **直接打开** - 双击 `index.html` 文件在浏览器中打开
2. **本地服务器** - 使用 Live Server 等工具运行

## 交互功能

- **点击屏幕** - 有缩放动画效果
- **悬停护腕** - 护腕会轻微放大
- **空格键** - 暂停/恢复屏幕动画
- **触摸支持** - 手机端触摸屏幕有反馈

## 技术栈

- HTML5
- CSS3 (渐变、阴影、动画)
- JavaScript (ES6+)
- 响应式设计
